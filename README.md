# MTR Connection Monitor

## Overview
MTR (My Traceroute) is a network diagnostic tool that combines the functionality of ping and traceroute.
This add-on provides enhanced monitoring capabilities for MTR connections.

## Features
- **Real-time monitoring** of network connections.
- Comprehensive reporting of packet loss and connectivity issues.
- Integration with existing network management tools.

## Installation
To install the MTR Connection Monitor add-on:
1. Clone the repository:
   ```
   git clone https://github.com/housemaister/mtr-monitor-add-on.git
   ```
2. Follow the setup instructions in the repository.

## Usage
Once installed, run the connection monitor using the following command:
```bash
mtr -c 100 -r <target>
```
Replace `<target>` with the desired host or IP address.
